# 배포 실행 안내

GitHub Pages 기준으로 배포하려면:

1) GitHub 저장소 초기화(최초 1회)
```bash
git init
git add .
git commit -m "feat: supplement advisor prototype"
git branch -M main
git remote add origin https://github.com/<your-id>/<your-repo>.git
git push -u origin main
```

2) GitHub에서 Pages가 활성화되어 있는지 확인
- Repository Settings > Pages > Source: GitHub Actions

3) 매 푸시마다 자동 배포
- `.github/workflows/deploy-supplement-pages.yml`가 `supplement-advisor` 폴더를 배포

배포 URL 예시:
- `https://<your-id>.github.io/<your-repo>/`

참고: 이미 Repository가 있고 Push만 필요하다면 `git add/commit/push`만 하면 됩니다.
